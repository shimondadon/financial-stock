# מדריך שימוש במערכת Cache - Local Files או MongoDB Cloud

המערכת תומכת בשתי שיטות אחסון cache:
1. **קבצים מקומיים (Local Files)** - ברירת המחדל, פשוט ומהיר
2. **MongoDB Cloud** - אחסון בענן, נגיש מכל מקום

## איך לבחור?

פתח את הקובץ `.env` ושנה את הערך של `CACHE_TYPE`:

### אפשרות 1: שימוש בקבצים מקומיים (ברירת מחדל)

```env
CACHE_TYPE=file
```

**יתרונות:**
- ✅ פשוט להתקנה - לא דורש הגדרה נוספת
- ✅ מהיר - אין צורך בחיבור לאינטרנט
- ✅ עובד מיד ללא הגדרות
- ✅ טוב למחשב יחיד

**החסרונות:**
- ❌ הנתונים נשמרים רק במחשב המקומי
- ❌ לא נגיש ממחשבים אחרים

### אפשרות 2: שימוש ב-MongoDB Cloud

```env
CACHE_TYPE=mongodb
MONGODB_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
```

**יתרונות:**
- ✅ נגיש מכל מחשב
- ✅ גיבוי אוטומטי
- ✅ ניתן לראות את הנתונים בממשק גרפי
- ✅ חינמי עד 512MB

**החסרונות:**
- ❌ דורש הגדרה של MongoDB Atlas
- ❌ דורש חיבור לאינטרנט

## הוראות שימוש מהירות

### כרגע: קבצים מקומיים (מומלץ להתחלה)

אם לא שינית כלום בקובץ `.env`, המערכת כבר עובדת עם קבצים מקומיים!

פשוט הרץ:
```bash
node server.js
```

תראה:
```
📁 Cache Type: Local Files
✅ Using local file-based cache
```

הנתונים נשמרים בתיקייה `cache/` בפרויקט.

### מעבר ל-MongoDB (אופציונלי)

אם אתה רוצה להשתמש ב-MongoDB Cloud:

1. עקוב אחרי המדריך המלא ב-`MONGODB_SETUP.md`
2. שנה בקובץ `.env`:
   ```env
   CACHE_TYPE=mongodb
   MONGODB_URI=mongodb+srv://your-username:your-password@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
   ```
3. הפעל מחדש את השרת

תראה:
```
📊 Cache Type: MongoDB Cloud
✅ Successfully connected to MongoDB
```

## החלפה בין השיטות

אפשר להחליף בכל רגע! פשוט:

1. עצור את השרת (Ctrl+C)
2. שנה את `CACHE_TYPE` בקובץ `.env`
3. הפעל מחדש: `node server.js`

**שים לב:** הנתונים לא מועתקים אוטומטית בין השיטות. כל שיטה שומרת את הנתונים שלה בנפרד.

## איזו שיטה מתאימה לי?

### השתמש בקבצים מקומיים אם:
- 📌 אתה עובד רק על מחשב אחד
- 📌 אתה רוצה פשטות מקסימלית
- 📌 אין לך צורך בגישה מרחוק

### השתמש ב-MongoDB אם:
- 📌 אתה עובד ממספר מחשבים
- 📌 אתה רוצה גיבוי בענן
- 📌 אתה רוצה לראות את הנתונים בממשק גרפי
- 📌 אתה רוצה לשתף נתונים עם אחרים

## מצב נוכחי

בדוק את הקובץ `.env` שלך:
- אם `CACHE_TYPE=file` או לא מוגדר → קבצים מקומיים ✅
- אם `CACHE_TYPE=mongodb` → MongoDB Cloud (צריך להגדיר MONGODB_URI)

---

**המלצה:** התחל עם קבצים מקומיים (`file`). אם תרצה אחר כך, תוכל בקלות לעבור ל-MongoDB.
